Introduction to HTML

HTML(HyperText Markup Language) is the code that is used to structure a webpage an its content.
For example, content could be structured within a set of paragraphs, a list of bulleted points or using images and data tables

So What is HTML?

HTML is a markup language that defines the structure of your content. it consists of a series elements which you use to enclose, or wrap different parts of the content to make it appear or act a certain way. Anatomy of an HTML element

The main parts of an HTML element are as follows:

  1. The opening tag: This consists of the name of the element(e.g p) wrapped in opening and closing angle brackets. This states where the element begins or starts to take effect.
  2. The closing tag: This is the same as the closing tag, except that it includes a forward slash before the element name. This states where the element ends- in this case where the paragraph ends.
  3. The content: This is the content of the element, which in this case is just text.
  4. The element: The opening tag, the closing tag, and the content together comprise the element.

Summary

HTML is a markup language used to describe the structure and content of a webpage. it says what content should be on webpage. An HTML document is made up of a series of elements. Some elements we covered in this lesson are:

  • heading tag
  • paragraph
  • list tags
  • image
  • hyper references(link)
